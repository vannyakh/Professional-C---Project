## Building the Project

### Prerequisites
- CMake (version 3.10 or higher)
- C++ compiler with C++17 support

### Build Instructions
1. Create a build directory:
```bash
mkdir build
cd build
```

2. Configure the project:
```bash
cmake ..
```

3. Build the project:
```bash
cmake --build .
```
